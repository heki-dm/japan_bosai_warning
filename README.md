# japan_bosai_warning

Flutter plugin for disaster prevention information in Japan.
## Sample
### Area
Get Area data.

```dart 
import 'package:japan_bosai_warning/japan_bosai_warning.dart';

final Area? areas =await getArea();
print(areas); 
```
